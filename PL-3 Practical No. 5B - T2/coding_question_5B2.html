Problem Statement: Write a program in any language to read 10 numbers fromkeyboard and find their sum and average.

             Program

#include < bits/stdc++.h >
void main()
{
      int arr[10], sum = 0;
      float avg;
      for(int i=0; i<10; i++){
        cin>>arr[i];
        sum += arr[i];
      }

      avg = sum/10.0;
      cout<<"Sum of array element : "<< sum<<"\n";
      cout<<"Average of array elements : "<<avg<<"\n";
      return 0;
}

Input

10 Interger values

Output

Sum and average of given 10 elements

Variables used in the program

arr, sum, avg

Complexity

The complexitiy of above program is: O(n)
